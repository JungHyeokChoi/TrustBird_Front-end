## 2. Get Start

### * Pre-Setting

- Docker
- Node.js(ver 12.x)
- Go
- npm
- TrustBrid_Back-end



### a. Setting .env

```
REACT_APP_SERVER => TrustBird_Back-end Address (ex. http://localhost:3001)
REACT_APP_IPFS => TrustBird_Back-end Address (ex. http://localhost:8080)
```



### b. npm install

```
npm install || npm i
```



### c. npm start

```
npm start
```
